二:查看状态

 1： 
   git status 是可以查看你的文件夹里文件的状态，

   会显示 文件是否被追踪，也就是这个文件是否在缓存区里，

   当缓存区的文件 有被更改过，会提示 modified，

   提示你把最近的修改，git add 提交。



 2：
   git log 会把你的 git commit 打印出来，

   你可以看到每次提交的信息，是谁提交了，

  以及提交时 -m 写入的文字，

  有时候会很多，可以用  git log --oneline

  简化信息，你可以更方便查看

  
  3：还有很多的命令行去查看状态，我们只记住简单的
